    const x1=cx+Math.cos(ang)*(r-10), y1=cy+Math.sin(ang)*(r-10);

    const x2=cx+Math.cos(ang)*(r-20), y2=cy+Math.sin(ang)*(r-20);

  ctx.beginPath(); ctx.moveTo(cx,cy); ctx.lineTo(cx+Math.cos(minAng)*(r-25), cy+Math.sin(minAng)*(r-25)); ctx.stroke();

  // center

  ctx.fillStyle="#0f3554"; ctx.beginPath(); ctx.arc(cx,cy,4,0,Math.PI*2); ctx.fill();

}

2) Clock Trainer

- Random target time in 5-minute steps (HH:MM, 12-hour).

- Sliders: Hour (1–12) and Minute (0…55 step 5); canvas analog clock updates live.

- “Check” compares to target; track correct count & attempts.

 

3) Fraction Pizza

- Random fraction numerators/denominators (denominator ∈ {2,3,4,5,6,8,10,12}, 1 ≤ numer < denom).

- Canvas pizza divided into `denom` slices; user shades slices via “＋/−”.

- “Check” validates; track correct & attempts.
